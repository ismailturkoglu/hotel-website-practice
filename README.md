## Proje Hakkında

Bu proje:

- Responsive tasarım içerir
- Flexbox yapısı kullanır
- Çoklu sayfa yapısına sahiptir (index & blogs)
- Modern ve sade bir UI hedeflenmiştir

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- Flexbox
- Responsive Design
